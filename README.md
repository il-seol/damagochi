# damagochi
A 10-stage fixed-width dot-art Tamagotchi that grows by syncing in real-time with your PC environment (CPU usage and system time). It features a smooth, distortion-free "bounce-in-place" animation and a dynamic weather-based background system.

## ✨ Key Features
- 10-Stage Evolution Tree: Watch your pet grow from a mysterious data egg into the Ultimate Slime God based on specific conditions.
- Real-Time Weather Sync: Automatically detects the "weather" (☀️ Clear, ☁️ Cloudy, ⚡ Busy) based on CPU usage and dynamically changes the background color.
- Day & Night Themes: Seamlessly adjusts the background opacity to match the actual system time.
- User-Friendly Controls: Supports mouse drag-and-drop movement, mouse wheel scrolling for resizing, and right-click menus for manual opacity control.

## 🚀 How to Run

### 1) Running via Source Code
Ensure Python is installed on your system, then install the required libraries and run the script:

pip install PyQt5 psutil
python pet.py
